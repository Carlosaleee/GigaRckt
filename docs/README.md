# 🚀 Landing Page - Giga Rckt - Agência de Eventos Corporativos

Projeto de uma landing page moderna e interativa desenvolvida para apresentação de serviços de uma agência de eventos corporativos.

---

## 🧠 Sobre o Projeto

Este projeto foi desenvolvido com foco em:

- Apresentação visual de alto impacto  
- Experiência do usuário (UX)  
- Navegação fluida e intuitiva  
- Estrutura moderna de landing page  

A proposta é transformar um material estático (PDF) em uma experiência digital interativa, mantendo fidelidade visual e adicionando interatividade ao conteúdo.

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla)  

---

## 🎯 Funcionalidades

- ✅ Navegação com menu fixo e responsivo  
- ✅ Scroll suave entre seções  
- ✅ Animações ao rolar a página (Intersection Observer)  
- ✅ Menu responsivo (mobile)  
- ✅ Navbar inteligente (oculta ao descer e exibe ao subir)  
- ✅ Botão "voltar ao topo"  
- ✅ Footer interativo com links de contato  
- ✅ Layout baseado em imagens (PDF convertido)
